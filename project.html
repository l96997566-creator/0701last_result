04 · Process
개발 순서
1

레이아웃 설계

헤더, 영화 그리드, 모달 구조를 먼저 잡고 다크 테마 CSS 기본 스타일 작성

2

TMDB API 연동

인기 영화 목록 fetch → 카드 렌더링 → 장르 목록 불러와 필터 버튼 생성

3

검색 & 상세 모달

디바운스 검색 구현, 영화 클릭 시 줄거리·평점·장르·예고편·유사 영화 표시

4

댓글 기능

영화 ID를 키로 localStorage에 댓글 저장·삭제 구현

5

배포

GitHub Add file 업로드 후 GitHub Pages 활성화해 링크 제출

05 · Update

NEW

예고편 관람 여부 표시

→ localStorage · dataset · classList 활용
